DES ECB解密。只能解密大小为8字节的数据。

参数:
const void* key, 解密用的密钥，和加密用的密钥相同。key指向的内存必须>=8个字节，并且只有前8个字节有效。
const void* s_text, 需要解密的数据。s_text指向的内存必须>=8个字节，并且只有前8个字节被解密。
void* d_text, 解密后的明文(8个字节)会存放到d_text里。
返回:
void
